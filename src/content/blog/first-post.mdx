# Getting Started with NixOS and Hyprland

If you're looking to build a **modern Linux desktop** with cutting-edge Wayland technology, NixOS and Hyprland make an excellent combination[web:21][web:25].

## Why Choose This Stack?

NixOS provides declarative system configuration, making your entire setup reproducible. Combined with Hyprland's tiling window manager capabilities, you get a powerful development environment[web:21][web:23].
